cd = change directory
cd .. = go back
mkdir = create folder
touch = create file
mv = rename

1. ls - відображає список файлів та каталогів у поточній директорії.
2. cd - змінює поточну директорію.
3. pwd - виводить повний шлях поточної директорії.
4. mkdir - створює новий каталог.
5. rm - видаляє файли або каталоги.
6. cp - копіює файли та каталоги.
7. mv - переміщає або перейменовує файли та каталоги.
8. touch - створює новий файл або оновлює час доступу та модифікації існуючого файлу.
9. cat - виводить вміст файлу.
10. less - дозволяє переглядати вміст файлу посторінково.
11. head - виводить перші рядки файлу.
12. tail - виводить останні рядки файлу.
13. grep - шукає заданий текст у файлах або виведення команд.
14. find - знаходить файли та каталоги на основі різних критеріїв.
15. chmod - змінює права доступу до файлів та каталогів.
16. chown - змінює власника файлів і каталогів.
17. chgrp - змінює групу файлів та каталогів.
18. tar - створює або розпаковує архіви.
19. zip - створює ZIP-архіви.
20. unzip - витягує файли з ZIP-архівів.
